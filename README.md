   - Wyniki kwalifikacji
   - Ranking z FP1 
   - Ranking z FP2
   - Ranking z FP3
   - Tempo wyścigowe
   - Temperatura toru
   - Temperatura powietrza
   - Charakterystyka toru (średnia prędkość, liczba wolnych zakrętów, klasyfikacje bolidów z poprzednich lat)
   - Marka silnika
   - Top speed z FP1/FP2/FP3/QUALI
